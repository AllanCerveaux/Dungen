# Dungen

Simple generator dungeon in typescript.

## Installation
### Requirements
- node: `>=18.18.2`

**Package Manager**
- npm: `>=10.2.1`
- pnpm: `>=8.9.2`
- yarn: `>=4.0.0`


```bash
git clone git@github.com:AllanCerveaux/Dungen.git
[npm|pnpm|yarn] install
```
